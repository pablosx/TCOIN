# Contributing to TCOIN

Thank you for your interest in contributing to TCOIN! By contributing, you agree to follow these guidelines to keep the project secure, clear, and maintainable.

## How to Contribute

### 1. Reporting Issues
- Use [GitHub Issues](https://github.com/pablosx/TCOIN/issues) to report bugs or suggest features.
- Include as much detail as possible:
  - Steps to reproduce
  - Node version and setup
  - Screenshots or logs if applicable
- Follow existing issue labels if possible.

### 2. Pull Requests
- Fork the repository.
- Create a branch for your feature or fix:
  ```bash
  git checkout -b my-feature-branch
